# 防災ブログをはじめました

はじめまして。このブログでは、**家庭や地域ですぐ役立つ防災情報** を、やさしい言葉でお届けしていきます。

## このブログで書いていくこと

- 地震・台風・大雨など、災害ごとの備えのポイント
- 非常持ち出し袋・備蓄のチェックリスト
- 高齢者やお子さんがいるご家庭向けの注意点
- 自治体や公的機関の最新情報の読み解き方

## まずは「3日分の水と食料」から

最初の備えとして、**ひとり1日3リットルの水を3日分** を目安にしてみてください。
4人家族なら、2リットルのペットボトル6本入りを3ケースが目安です。

![水のペットボトルのイメージ](images/sample-water.svg)

> 「完璧に揃えてから」ではなく、「今日できる1つ」から始めるのがコツです。

## 記事内に画像を入れる方法

記事のMarkdown内に画像を入れたいときは、次のように書きます：

```
![画像の説明文](images/ファイル名.jpg)
```

`images/` フォルダに画像ファイルを置いておけば、自動で表示されます。

## アフィリエイトリンクの貼り方（参考）

アフィリエイトリンクは、本文中に普通のリンクとして貼れます：

```
[防災セットを楽天で見る](https://example.com/xxxxx)
```

「ここを目立たせたい！」というときは、ボックス型のリンクを使うこともできます：

```
<a class="affiliate-box" href="https://example.com/xxxxx" target="_blank" rel="nofollow sponsored">
  🛒 おすすめの防災セットを見る
</a>
```

これからどうぞよろしくお願いします。
